Oxy Students' Attitudes Toward Various Topics

Dates: July 2019 - August 2019

Why I made this: "Oxy Confessions", an anonymous Facebook page started by Oxy students in October of 2013, contains 34,000+ posts consisting of gossip, strife, and honest opinions about worldly topics. Inspired by Ben Schmidt's Gendered Language in Teacher Reviews, I made this website to reveal Oxy students' attitudes toward various topics in the world.

Tools used: Node.js, MySQL, HTML, CSS, JavaScript, Chart.js

NBA Clutch Project

Dates: December 2019 - January 2020

Why I made this: Everyone has an answer, but no one ever has enough evidence to back it up.

... Yes, I'm talking about comparing the clutchness of NBA players. To accurately evaluate one's clutchness, a wealth of data across long periods of time is needed. I built this web app to allow for the comparison of NBA players under different game scenarios, equiping users with the tools to settle this age-old debate. It pulls data directly from stats.nba.com's API, ensuring that every graph 100% accurate.

Tools used: Flask, nba_api, HTML, CSS, JavaScript, Chart.js
